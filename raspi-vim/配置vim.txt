安装插件
1.pydiction自动补全插件

自动补全插件Pydiction是一款好用的Vim Python主动补全插件，首要便利用于在vim中运用tab键主动补全Python代码，支撑输入部分Python关键字、模块名、特点和方法名等，兼容规范的、自定义的和第三方模块和包。

安装Pydiction：
cd ~/.vim/plugged
git clone https://github.com/rkulla/pydiction.git
cp -r ~/.vim/plugged/pydiction/after/ ~/.vim

配色方案：
mkdir ~/.vim/color
cd ~/.vim/color
